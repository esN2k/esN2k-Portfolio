# Changelog

- Repository hygiene files: `.editorconfig`, `.gitattributes`, `.gitignore`

- Optimized the main profile image for faster page loads
- Simplified the project into a clean static GitHub Pages site

### Removed

- Legacy Flutter web build artifacts and unused bundled assets
- Unused third-party font dependency from Google Fonts
